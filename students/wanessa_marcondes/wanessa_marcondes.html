Meus outros aprendizados e interesses

{ Criação de conteúdo }

Em 2017 desenvolvi um projeto acadêmico voltado para a busca de terapias alternativas no tratamento de transtornos psicológicos, com foco em estudantes universitários. Em equipe, desenvolvemos um aplicativo móvel que apresenta receitas voltadas para o alívio de emoções negativas, acompanhado do projeto de um canal no Youtube com conversas sobre assuntos como ansiedade e ensinando a cocção das receitas do aplicativo.

{ Trabalho voluntário }

Este ano, novamente em grupo acadêmico, participei do desenvolvimento de uma plataforma para divulgar e incentivar o trabalho voluntário voltado para idosos. Utilizando dos recursos de gamificação, o projeto visa recompensar o voluntário com badges e certificados prontos para serem compartilhados nas redes sociais e LinkedIn.

{ Realidade Virtual e Internet das Coisas }

Novamente com foco na saúde mental dos jovens, lançamos um aplicativo integrado com realidade virtual (VR) e Arduino. O pacote para a experiência completa do aplicativo inclui óculos VR e uma pulseira, conectada com o aplicativo, capaz de identificar uma possível crise de ansiedade. Ao identificar a crise, o aplicativo notifica o usuário e sugere que assista um vídeo em 360 VR para se acalmar.
